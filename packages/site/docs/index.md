# rinhany

**Motor extensível de rinha de algoritmos.**

Toda rinha reinventa sua própria infraestrutura. **rinhany** é o motor que falta. Você traz o problema, os algoritmos e as métricas — rinhany cuida do ciclo de execução, dos rankings e da apresentação.

## Pacotes

| Pacote | Descrição |
|--------|-----------|
| `rinhany` | Meta-pacote que re-exporta tudo |
| `@rinhany/core` | Engine: executor, ringue, types, juiz e logger |
| `@rinhany/cli` | UI de terminal: apresentador, narrador, público e tema |
| `@rinhany/testing` | Utilities de teste e contract tests para algoritmos |
